共享组件架构

消除重复，所有界面使用同一套组件

1. 统一对话框 (UnifiedGameDialog)

替代现有的 HalfScreenDialog + 私有 CommonDialog + Dialog(usePlatformDefaultWidth=false) 三种模式，统一为一个。

模式宽度高度使用场景
HalffillMaxWidth(0.85f)fillMaxHeight(0.78f)建筑/生产界面 (默认)
FullfillMaxSizefillMaxSize弟子/仓库/设置
AutofillMaxWidth(0.85f)wrapContentHeight简短确认/信息弹窗

三种模式共享：bg_horizontal 背景、12dp 圆角、头部结构、关闭按钮位置

2. 统一插槽组件 (UnifiedSlot + SlotGrid)

替代各文件中独立定义的 ElderSlotItem、DirectDiscipleSlot、ProductionSlotItem、MinerSlot 等。

变体尺寸 Token用法
UnifiedSlot.Large70dp生产建筑长老位
UnifiedSlot.Medium60dp峰长老 / 生产槽 / 矿工位 / 监牢
UnifiedSlot.Small52dp亲传弟子 / 传道师 / 执法弟子

此外提供 SlotRow 和 SlotGrid 布局容器，统一每行排列方式和间距 (8dp)。

3. 统一筛选器 (DiscipleFilterBar)

将 SpiritRootAttributeFilterBar + RealmFilterRow 及其 applyFilters 逻辑合并为一个组件。消除 10+ 处重复。

参数化：realmSet (包含哪些境界)、showSpiritRootFilter、showAttributeSort、showRealmFilter 均为可选开关。

影响文件: ProductionComponents, PeakScreenComponents, LibraryScreen, SpiritMineScreen, AllianceScreen, MerchantScreen, RecruitScreen

4. 统一空状态 (EmptyState)

替代各处内联的 "暂无XXX" Box 写法，统一为居中文字 + 可选图标。

使用 Body(11sp) 字体，高度 120dp，居中。
