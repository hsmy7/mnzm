# 灵药园灵植属性成熟速度加成 Spec

## Why
灵药园中灵植长老和亲传弟子的灵植属性应该影响种子成熟速度，增加游戏的策略深度，让玩家在任命灵植职位时需要考虑灵植属性。

## What Changes
- 灵植长老的灵植属性会影响种子成熟速度：以50为基准，每高5点增加1%成熟速度，每低5点减少1%成熟速度
- 亲传弟子的灵植属性会影响种子成熟速度：以50为基准，每高10点增加1%成熟速度，每低10点减少1%成熟速度

## Impact
- Affected specs: 灵药园系统
- Affected code:
  - `GameEngine.kt` - `calculateElderAndDisciplesBonus()` 方法中灵药园部分的成熟时间减少计算逻辑

## ADDED Requirements

### Requirement: 灵植长老灵植属性成熟速度加成
系统应当让灵植长老的灵植属性对种子成熟速度提供加成或减益。

#### Scenario: 灵植长老灵植属性高于50
- **WHEN** 灵植长老的灵植属性为60
- **THEN** 种子成熟速度增加2%（(60-50)/5 * 1% = 2%），即成熟时间减少2%

#### Scenario: 灵植长老灵植属性低于50
- **WHEN** 灵植长老的灵植属性为40
- **THEN** 种子成熟速度减少2%（(40-50)/5 * 1% = -2%），即成熟时间增加2%

#### Scenario: 灵植长老灵植属性等于50
- **WHEN** 灵植长老的灵植属性为50
- **THEN** 种子成熟速度不变（(50-50)/5 * 1% = 0%）

#### Scenario: 无灵植长老
- **WHEN** 没有任命灵植长老
- **THEN** 种子成熟速度不受此加成影响

### Requirement: 亲传弟子灵植属性成熟速度加成
系统应当让亲传弟子的灵植属性对种子成熟速度提供加成或减益。

#### Scenario: 亲传弟子灵植属性高于50
- **WHEN** 亲传弟子的灵植属性为70
- **THEN** 种子成熟速度增加2%（(70-50)/10 * 1% = 2%），即成熟时间减少2%

#### Scenario: 亲传弟子灵植属性低于50
- **WHEN** 亲传弟子的灵植属性为30
- **THEN** 种子成熟速度减少2%（(30-50)/10 * 1% = -2%），即成熟时间增加2%

#### Scenario: 亲传弟子灵植属性等于50
- **WHEN** 亲传弟子的灵植属性为50
- **THEN** 种子成熟速度不变（(50-50)/10 * 1% = 0%）

#### Scenario: 多个亲传弟子
- **WHEN** 有多个亲传弟子任职
- **THEN** 每个亲传弟子的加成独立计算并叠加

## MODIFIED Requirements

### Requirement: 灵药园成熟时间减少计算公式
修改 `calculateElderAndDisciplesBonus` 方法中灵药园部分的成熟时间减少计算：

**修改前**：
- 长老：每多5点减少1%成熟时间（只有正向）
- 亲传弟子：每多10点减少1%成熟时间（只有正向）

**修改后**：
- 长老：以50为基准，每高5点增加1%成熟速度，每低5点减少1%成熟速度
- 亲传弟子：以50为基准，每高10点增加1%成熟速度，每低10点减少1%成熟速度

## 计算公式
```
灵植长老成熟速度加成 = (灵植长老灵植属性 - 50) / 5 * 0.01
亲传弟子成熟速度加成 = (亲传弟子灵植属性 - 50) / 10 * 0.01
总成熟速度加成 = 灵植长老成熟速度加成 + 所有亲传弟子成熟速度加成之和
成熟时间减少比例 = 总成熟速度加成
```

例如：
- 灵植长老灵植属性 70：加成 = (70-50)/5*0.01 = 4% 成熟速度加成（成熟时间减少4%）
- 灵植长老灵植属性 30：加成 = (30-50)/5*0.01 = -4% 成熟速度减益（成熟时间增加4%）
- 亲传弟子灵植属性 70：加成 = (70-50)/10*0.01 = 2% 成熟速度加成
- 亲传弟子灵植属性 30：加成 = (30-50)/10*0.01 = -2% 成熟速度减益

## 限制
- 成熟时间减少比例上限为50%（防止成熟时间过短）
- 成熟时间减少比例下限为-30%（防止成熟时间过长）
