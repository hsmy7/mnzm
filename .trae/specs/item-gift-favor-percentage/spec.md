# 物品送礼好感度百分比增长机制规格

## Why

当前物品送礼系统使用固定好感度增加（凡品+1、灵品+5、宝品+15、玄品+25、地品+35、天品+50），导致高好感度宗门送礼效果不明显，低好感度宗门送礼效果过于显著。需要改为按当前好感度的百分比增长，使送礼效果更加合理且与宗门等级相关，送的越多加的越多。

## What Changes

- **BREAKING** 修改物品送礼好感度计算公式：从固定值改为当前好感度的百分比增长
- 新增宗门等级与物品稀有度对应的好感度百分比配置
- 不同宗门等级接受不同稀有度物品的好感度增长百分比不同

## Impact

- Affected specs: sect-gift-favor-system
- Affected code: GiftConfig.kt, GameEngine.kt

## ADDED Requirements

### Requirement: 物品送礼好感度百分比增长配置

系统应根据宗门等级和物品稀有度计算好感度增长百分比。

#### Scenario: 小型宗门好感度增长
- **GIVEN** 小型宗门（level=0）
- **WHEN** 送凡品物品
- **THEN** 好感度增加当前好感度的30%
- **WHEN** 送灵品物品
- **THEN** 好感度增加当前好感度的60%
- **WHEN** 送宝品物品
- **THEN** 好感度增加当前好感度的200%
- **WHEN** 送玄品物品
- **THEN** 好感度增加当前好感度的300%
- **WHEN** 送地品物品
- **THEN** 好感度增加当前好感度的400%
- **WHEN** 送天品物品
- **THEN** 好感度增加当前好感度的500%

#### Scenario: 中型宗门好感度增长
- **GIVEN** 中型宗门（level=1）
- **WHEN** 送凡品物品
- **THEN** 好感度增加当前好感度的20%
- **WHEN** 送灵品物品
- **THEN** 好感度增加当前好感度的50%
- **WHEN** 送宝品物品
- **THEN** 好感度增加当前好感度的100%
- **WHEN** 送玄品物品
- **THEN** 好感度增加当前好感度的150%
- **WHEN** 送地品物品
- **THEN** 好感度增加当前好感度的200%
- **WHEN** 送天品物品
- **THEN** 好感度增加当前好感度的400%

#### Scenario: 大型宗门好感度增长
- **GIVEN** 大型宗门（level=2）
- **WHEN** 送玄品物品
- **THEN** 好感度增加当前好感度的40%
- **WHEN** 送地品物品
- **THEN** 好感度增加当前好感度的100%
- **WHEN** 送天品物品
- **THEN** 好感度增加当前好感度的200%
- **WHEN** 送凡品/灵品/宝品物品
- **THEN** 被拒绝（现有拒绝机制已覆盖）

#### Scenario: 顶级宗门好感度增长
- **GIVEN** 顶级宗门（level=3）
- **WHEN** 送玄品物品
- **THEN** 好感度增加当前好感度的30%
- **WHEN** 送地品物品
- **THEN** 好感度增加当前好感度的80%
- **WHEN** 送天品物品
- **THEN** 好感度增加当前好感度的160%
- **WHEN** 送凡品/灵品/宝品物品
- **THEN** 被拒绝（现有拒绝机制已覆盖）

### Requirement: 好感度计算公式

系统应使用百分比公式计算最终好感度。

#### Scenario: 好感度计算
- **GIVEN** 宗门当前好感度为C，物品稀有度对应百分比为P%
- **WHEN** 送礼成功
- **THEN** 新好感度 = min(C + C * P / 100, 100)

#### Scenario: 好感度下限保护
- **GIVEN** 宗门当前好感度为0
- **WHEN** 送礼成功
- **THEN** 好感度至少增加1点（避免0 * 百分比 = 0的情况）

#### Scenario: 多数量物品送礼
- **GIVEN** 送N个相同物品
- **WHEN** 送礼成功
- **THEN** 总好感度 = 单个物品好感度 * N

## MODIFIED Requirements

### Requirement: 物品稀有度好感度配置修改

原有物品稀有度好感度配置使用固定值，现修改为百分比配置。

#### Scenario: 配置结构变更
- **GIVEN** RarityFavor数据类
- **WHEN** 修改配置
- **THEN** 移除固定favor字段，改为按宗门等级配置的百分比映射

## REMOVED Requirements

### Requirement: 固定好感度值配置

**Reason**: 改为百分比增长机制，不再使用固定值
**Migration**: 将GiftConfig中的RarityFavorConfig改为百分比配置

## 核心数值表

### 好感度增长百分比（按宗门等级和物品稀有度）

| 宗门等级 | 凡品(1) | 灵品(2) | 宝品(3) | 玄品(4) | 地品(5) | 天品(6) |
|---------|--------|--------|--------|--------|--------|--------|
| 小型宗门(0) | 30% | 60% | 200% | 300% | 400% | 500% |
| 中型宗门(1) | 20% | 50% | 100% | 150% | 200% | 400% |
| 大型宗门(2) | - | - | - | 40% | 100% | 200% |
| 顶级宗门(3) | - | - | - | 30% | 80% | 160% |

注： "-" 表示该稀有度会被该等级宗门拒绝

### 好感度计算示例

假设当前好感度为20：

| 宗门等级 | 凡品后 | 灵品后 | 宝品后 | 玄品后 | 地品后 | 天品后 |
|---------|--------|--------|--------|--------|--------|--------|
| 小型宗门 | 26(+6) | 32(+12) | 60(+40) | 80(+60) | 100(+80) | 100(+80) |
| 中型宗门 | 24(+4) | 30(+10) | 40(+20) | 50(+30) | 60(+40) | 100(+80) |
| 大型宗门 | - | - | - | 28(+8) | 40(+20) | 60(+40) |
| 顶级宗门 | - | - | - | 26(+6) | 36(+16) | 52(+32) |

### 配置数据结构

// 物品送礼好感度百分比配置：key为宗门等级，value为稀有度到百分比的映射

    // 小型宗门
    0 to mapOf(
        1 to 30,   // 凡品
        2 to 60,   // 灵品
        3 to 200,  // 宝品
        4 to 300,  // 玄品
        5 to 400,  // 地品

    // 大型宗门

        4 to 40,   // 玄品
