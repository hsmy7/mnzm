# 天工峰炼器属性速度加成调整规范

## Why
当前天工峰长老和亲传弟子的炼器属性对炼器速度的加成比例过高，需要调整为更合理的数值，使游戏平衡性更好。

## What Changes
- 调整天工长老炼器属性对炼器速度的加成：从每1点偏离50增加1%，改为每5点偏离50增加1%
- 调整亲传弟子炼器属性对炼器速度的加成：从每5点偏离50增加1%，改为每10点偏离50增加1%
- 保留成功率加成不变（长老每5点增加1%，亲传弟子每10点增加1%）

## Impact
- Affected specs: 天工峰炼器系统
- Affected code:
  - `GameEngine.kt` - `calculateElderAndDisciplesBonus()` 方法中的 "forge" 分支

## ADDED Requirements

### Requirement: 天工长老炼器属性速度加成
系统应当根据天工长老的炼器属性提供炼器速度加成或减益。

#### Scenario: 天工长老炼器属性高于50
- **WHEN** 天工长老的炼器属性为60
- **THEN** 炼器速度增加2%（(60-50)/5 * 1% = 2%）

#### Scenario: 天工长老炼器属性低于50
- **WHEN** 天工长老的炼器属性为40
- **THEN** 炼器速度减少2%（(40-50)/5 * 1% = -2%）

#### Scenario: 天工长老炼器属性等于50
- **WHEN** 天工长老的炼器属性为50
- **THEN** 炼器速度不变（(50-50)/5 * 1% = 0%）

#### Scenario: 无天工长老
- **WHEN** 天工峰没有任命长老
- **THEN** 炼器速度不受此加成影响

### Requirement: 亲传弟子炼器属性速度加成
系统应当根据亲传弟子的炼器属性提供炼器速度加成或减益。

#### Scenario: 亲传弟子炼器属性高于50
- **WHEN** 亲传弟子的炼器属性为70
- **THEN** 炼器速度增加2%（(70-50)/10 * 1% = 2%）

#### Scenario: 亲传弟子炼器属性低于50
- **WHEN** 亲传弟子的炼器属性为30
- **THEN** 炼器速度减少2%（(30-50)/10 * 1% = -2%）

#### Scenario: 亲传弟子炼器属性等于50
- **WHEN** 亲传弟子的炼器属性为50
- **THEN** 炼器速度不变（(50-50)/10 * 1% = 0%）

## MODIFIED Requirements

### Requirement: 天工峰速度加成计算公式（修改）
原公式：
```
长老速度加成 = (炼器属性 - 50) * 0.01
亲传弟子速度加成 = (炼器属性 - 50) / 5.0 * 0.01
```

新公式：
```
长老速度加成 = (炼器属性 - 50) / 5.0 * 0.01
亲传弟子速度加成 = (炼器属性 - 50) / 10.0 * 0.01
```
