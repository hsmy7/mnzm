# 宗门新政策 Spec

## Why

现有宗门政策仅有2个（灵矿增产、增强治安），政策系统较为单薄。为了增加游戏策略深度和可玩性，需要扩展更多政策选项，让玩家在资源管理、弟子培养、设施效率、外交战斗等方面有更多选择。

## What Changes

* 新增5个宗门政策

* 修改灵矿增产政策（增加灵石消耗）

* 修改增强治安政策（调整效果和消耗）

* 扩展 `SectPolicies` 数据结构

* 新增政策效果逻辑实现

* 新增政策UI界面展示

## Impact

* Affected specs: 宗门政策系统

* Affected code:

  * `GameData.kt` - SectPolicies 数据结构

  * `GameEngine.kt` - 政策效果逻辑

  * `GameViewModel.kt` - 政策切换函数

  * `MainGameScreen.kt` - 政策UI界面

## 副宗主智力加成机制

所有政策的正面效果受副宗主智力影响：

* **基准值**：智力50为基准，无加成
* **加成规则**：每比50多5点智力，政策效果增加1%
* **示例**：副宗主智力70时，政策效果额外增加4%（(70-50)/5 = 4%）
* **注意**：若无副宗主，则无智力加成

### Requirement: SectPolicies 数据结构扩展

`SectPolicies` 数据结构需要扩展以支持新政策。

```kotlin
data class SectPolicies(
    // 已有政策
    val spiritMineBoost: Boolean = false,      // 灵矿增产
    val enhancedSecurity: Boolean = false,     // 增强治安
    
    // 资源生产类
    val alchemyIncentive: Boolean = false,     // 丹道激励
    val forgeIncentive: Boolean = false,       // 锻造激励
    val herbCultivation: Boolean = false,      // 灵药培育
    
    // 修炼提升类
    val cultivationSubsidy: Boolean = false,   // 修行津贴
    val manualResearch: Boolean = false        // 功法研习
)
```
