# 兑换码系统 Checklist

## 数据模型
- [x] RedeemCode 数据类包含所有必要属性（code、奖励类型、数量、使用次数限制、过期时间、启用状态）
- [x] RedeemRewardType 枚举包含所有奖励类型（灵石、装备、功法、丹药、材料、灵草、种子、弟子）
- [x] DiscipleRewardConfig 数据类包含弟子配置（境界、灵根类型、灵根数量、天赋、属性、年龄、性别）
- [x] GameData 中添加 usedRedeemCodes 字段记录已使用兑换码
