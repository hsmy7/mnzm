# 丹鼎殿炼丹属性速度加成 Spec

## Why
丹鼎殿中炼丹长老和亲传弟子的炼丹属性应该影响炼丹速度，增加游戏的策略深度，让玩家在任命炼丹职位时需要考虑炼丹属性。

## What Changes
- 炼丹长老的炼丹属性会影响炼丹速度：以50为基准，每高5点增加1%炼丹速度，每低5点减少1%炼丹速度
- 亲传弟子的炼丹属性会影响炼丹速度：以50为基准，每高10点增加1%炼丹速度，每低10点减少1%炼丹速度

## Impact
- Affected specs: 丹鼎殿系统
- Affected code:
  - `GameEngine.kt` - `calculateWorkDurationWithAllDisciples()` 方法需要使用长老槽位系统的速度加成
  - `GameEngine.kt` - `calculateElderAndDisciplesBonus()` 方法中炼丹部分的速度加成计算逻辑

## ADDED Requirements

### Requirement: 炼丹长老炼丹属性速度加成
系统应当让炼丹长老的炼丹属性对炼丹速度提供加成或减益。

#### Scenario: 炼丹长老炼丹属性高于50
- **WHEN** 炼丹长老的炼丹属性为60
- **THEN** 炼丹速度增加2%（(60-50)/5 * 1% = 2%）

#### Scenario: 炼丹长老炼丹属性低于50
- **WHEN** 炼丹长老的炼丹属性为40
- **THEN** 炼丹速度减少2%（(40-50)/5 * 1% = -2%）

#### Scenario: 炼丹长老炼丹属性等于50
- **WHEN** 炼丹长老的炼丹属性为50
- **THEN** 炼丹速度不变（(50-50)/5 * 1% = 0%）

#### Scenario: 无炼丹长老
- **WHEN** 没有任命炼丹长老
- **THEN** 炼丹速度不受此加成影响

### Requirement: 亲传弟子炼丹属性速度加成
系统应当让亲传弟子的炼丹属性对炼丹速度提供加成或减益。

#### Scenario: 亲传弟子炼丹属性高于50
- **WHEN** 亲传弟子的炼丹属性为70
- **THEN** 炼丹速度增加2%（(70-50)/10 * 1% = 2%）

#### Scenario: 亲传弟子炼丹属性低于50
- **WHEN** 亲传弟子的炼丹属性为30
- **THEN** 炼丹速度减少2%（(30-50)/10 * 1% = -2%）

#### Scenario: 亲传弟子炼丹属性等于50
- **WHEN** 亲传弟子的炼丹属性为50
- **THEN** 炼丹速度不变（(50-50)/10 * 1% = 0%）

#### Scenario: 多个亲传弟子
- **WHEN** 有多个亲传弟子任职
- **THEN** 每个亲传弟子的加成独立计算并叠加

## MODIFIED Requirements

### Requirement: 炼丹速度加成计算公式
修改 `calculateElderAndDisciplesBonus` 方法中炼丹部分的速度加成计算：

**修改前**：
- 长老：每多1点增加1%炼制速度
- 亲传弟子：每多5点增加1%炼制速度

**修改后**：
- 长老：每高5点增加1%炼制速度，每低5点减少1%炼制速度
- 亲传弟子：每高10点增加1%炼制速度，每低10点减少1%炼制速度

### Requirement: 炼丹速度加成应用
修改 `calculateWorkDurationWithAllDisciples` 方法，对于炼丹建筑使用 `calculateElderAndDisciplesBonus` 计算速度加成。

## 计算公式
```
炼丹长老速度加成 = (炼丹长老炼丹属性 - 50) / 5 * 0.01
亲传弟子速度加成 = (亲传弟子炼丹属性 - 50) / 10 * 0.01
总速度加成 = 炼丹长老速度加成 + 所有亲传弟子速度加成之和
```

例如：
- 炼丹长老炼丹属性 70：加成 = (70-50)/5*0.01 = 4% 炼丹速度加成
- 炼丹长老炼丹属性 30：加成 = (30-50)/5*0.01 = -4% 炼丹速度减益
- 亲传弟子炼丹属性 70：加成 = (70-50)/10*0.01 = 2% 炼丹速度加成
- 亲传弟子炼丹属性 30：加成 = (30-50)/10*0.01 = -2% 炼丹速度减益
