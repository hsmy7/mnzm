# 外门长老悟性加成外门弟子突破率 Spec

## Why
目前问道峰的外门长老职位没有实际功能，需要让外门长老的悟性对外门弟子的突破率提供加成，使该职位具有实际意义。

## What Changes
- 外门长老的悟性会影响外门弟子的突破率
- 悟性以50为基准，每高5点增加1%突破率，每低5点减少1%突破率
- 只有外门弟子（discipleType == "outer"）享受此加成

## Impact
- Affected specs: 问道峰系统
- Affected code:
  - `Disciple.kt` - `getBreakthroughChance()` 方法需要接收外门长老悟性参数
  - `GameEngine.kt` - `attemptBreakthrough()` 方法需要计算并传递外门长老悟性加成

## ADDED Requirements

### Requirement: 外门长老悟性加成外门弟子突破率
系统应当让问道峰外门长老的悟性对外门弟子的突破率提供加成或减益。

#### Scenario: 外门长老悟性高于50
- **WHEN** 外门弟子尝试突破境界
- **AND** 外门长老的悟性高于50
- **THEN** 外门弟子获得突破率加成（每高5点增加1%）

#### Scenario: 外门长老悟性低于50
- **WHEN** 外门弟子尝试突破境界
- **AND** 外门长老的悟性低于50
- **THEN** 外门弟子获得突破率减益（每低5点减少1%）

#### Scenario: 无外门长老
- **WHEN** 外门弟子尝试突破境界
- **AND** 没有任命外门长老
- **THEN** 外门弟子不获得任何加成或减益

#### Scenario: 外门长老已死亡
- **WHEN** 外门弟子尝试突破境界
- **AND** 外门长老已死亡
- **THEN** 外门弟子不获得任何加成或减益

#### Scenario: 非外门弟子
- **WHEN** 内门弟子或亲传弟子尝试突破境界
- **THEN** 不受外门长老悟性影响

## 计算公式
```
外门长老悟性加成 = (外门长老悟性 - 50) / 5 * 0.01
```

例如：
- 外门长老悟性 70：加成 = (70-50)/5*0.01 = 4% 突破率加成
- 外门长老悟性 30：加成 = (30-50)/5*0.01 = -4% 突破率减益
- 外门长老悟性 50：加成 = 0%（无影响）
